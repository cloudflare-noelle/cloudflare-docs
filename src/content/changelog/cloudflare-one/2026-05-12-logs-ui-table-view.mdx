Based on your feedback from the previous logs UI update, Zero Trust logs now feature an improved table-based format that makes it easier to scan and analyze your log data.

![Screenshot of the updated logs UI showing the table-based format with activity chart](~/assets/images/changelog/cloudflare-one/cf1-logs-ui-update-may2026.png)

The updated logs UI includes:

- **Table-based layout** - View your logs in a structured table format for easier scanning and comparison across entries.
- **Customizable columns** - Choose which columns to display in the table to focus on the data most relevant to you.
- **Expandable dropdowns** - Select any row to expand a dropdown and view the full details of a log entry without leaving the table view.
- **Activity chart** - A new chart visualizes allowed and blocked activity over time, helping you quickly identify trends and anomalies.

These updates apply to Access authentication logs and Gateway activity logs (DNS, Network, and HTTP).

For more information, refer to [Access authentication logs](/cloudflare-one/insights/logs/dashboard-logs/access-authentication-logs/) and [Gateway activity logs](/cloudflare-one/insights/logs/dashboard-logs/gateway-logs/).
